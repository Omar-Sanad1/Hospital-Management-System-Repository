# Hospital-Management-System-Repository
Hospital Management System API built with ASP.NET Core, Entity Framework Core, SQL Server, JWT Authentication, Role-Based Authorization,Filtering, Sorting, Pagination, AutoMapper, and Global Exception Handling.

- ASP.NET Core Web API

- Entity Framework Core
- SQL Server
